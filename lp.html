Q, WAO Let's Code pyって何？

WAO Let'sCode pyとは、京都大学発のITベンチャーDeMiAと協力し、

共通テスト形式「情報Ⅰ」の対策のために作られた「情報Ⅰ」の勉強に最適なアプリのことです！！

Let's Code pyの強み

・1問1答形式で、「情報Ⅰ」分野の知識を全て網羅！

・実際の共通テストを模した複数の予想問題で本番形式に沿った演習ができる！

・ブラウザ上でPythonのコードを実行できるエディター画面の提供！(※団体契約者様年用の提供)

どういう機能が受験生にとって最適なの？

01.

基本的に無料で「情報Ⅰ」の問題を解くことができる！

アプリをダウンロードするだけで一問一答の全問題、共通テスト形式の問題を
数問無料で解くことができる！

↓

・全ての受験生が、0円で「情報Ⅰ」の対策ができる!

・「情報Ⅰ」のIT用語・知識、それらを応用した論理的思考力を要する問題への対策がこれ一つで完結！

利用料

¥0

※ただし、アプリ内課金あり

02.

演習問題が豊富！！

共通テストに必要なアウトプットができる！

・一問一答の問題数はなんと計578問！

・共通テスト形式の問題は「ワオ・コーポレーション」により、質の高い問題をご用意！

↓

問題は適宜追加され、十分な演習と知識の定着化を図ることができる！

03.

空き時間に効率よく勉強することができる！！

スマホ一つで共通テスト「情報Ⅰ」の勉強が可能！

↓

電車での通学時間・受験勉強の空き時間などに
アプリを開くだけで「情報Ⅰ」の対策ができる！

↓

・「情報Ⅰ」を勉強する時間がない人

・他の苦手科目や二次試験対策に時間を使いたい人に特におすすめです！
